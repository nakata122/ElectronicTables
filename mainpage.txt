/*! \mainpage Electronic tables
 *
 * \section intro_sec Увод
 *
 * Проекта реализира клас Table, който управлява таблица.
 *
 * \section install_sec Тестове и примери
 *
 * \subsection step1 Отваряне на Table file
 *  open test1.txt \n
 *  Отваря примерен Table файл и го запазва в паметта. Извежда грешки, ако съществуват
 *  
 * \subsection step2 Принтиране на Table
 *  print \n
 *  Принтира таблицата подравнена на конзолата. Ако има полета формули се изчисляват наново
 * 
 * \subsection step3 Промяна на стойността на някоя клетка
 *  edit 0 0 = R0C1 * 2 + 5 - 3 * 4 ^ 2  \n
 *  edit 0 1 7  \n
 *  edit 0 0 "353"  \n
 *  edit 0 1 "463.23.42"  \n
 *  Променя стойността на базата на клетка на дадена позиция с произволен тип
 *
 * \subsection step4 Затваряне на Table object
 *  close
 *  Изтрива паметта за отворения файл(не самия файл)
 * 
 * \subsection step5 Запазване във файл
 *  save  \n
 *  saveas someFile.txt  \n
 *  Отпечатва информацията за таблицата, запазена от паметта във файл, в зависимост от параметрите
 */
